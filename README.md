# DDDCMS
Studying and learning management system, that includes and handles departments management, users management, courses management, skills management, session management, attendance and certification management. Using C#/ASP.NET Core, domain driven design, clean architecture, and many architectural patterns like CQRS, Mediator, and the Repository pattern. Needed NuGet packages are:

- ErrorOr
- MediatR
- Microsoft.EntityFrameworkCore
- Microsoft.DependencyInjection
- FluentValidation
- JwtSecurity
- Some others...
